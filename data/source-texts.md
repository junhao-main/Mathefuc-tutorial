# Source Texts

Source texts are the raw documents you read and annotate before extracting structured knowledge.

## Typical Structure

- A **text collection** acts like a folder
- Each **text item** is an individual source document
- **Source references** capture selections from those documents

## Best Practices

- Keep one topic per source item when possible
- Use descriptive titles for collections and items
- Commit annotations into well-named raw-material folders
