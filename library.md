# Library

Libraries help you organize definitions, theorems, propositions, and equations.

## Creating a Library

1. Go to **Data → Library**
2. Click **Create**
3. Name your library (e.g. "Real Analysis", "Linear Algebra")

## Adding Content

- Add math objects from the reference database
- Objects can appear in multiple libraries
- Use **Move** to reorganize between libraries

## Types of Math Objects

- **Definition** — Formal definitions
- **Theorem** — Statements with proofs
- **Proposition** — Supporting results
- **Equation** — Important formulas
