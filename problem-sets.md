# Problem Sets

Problem sets are collections of proof exercises. Use them to practice and track your progress.

## Creating a Problem Set

1. Go to **Data → Problem Sets**
2. Click **Create**
3. Enter a name and optional description
4. Add proofs from the reference library

## Adding Proofs

- From a problem set's detail page, click **Add proof**
- Search by title or category
- Proofs can belong to multiple problem sets

## Practicing

- Open a proof from the set
- Use the **IDE** to write your solution
- Submit to check your work (when available)
