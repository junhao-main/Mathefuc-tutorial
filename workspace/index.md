# Workspace

The workspace is the VSCode-style area for reading source texts, annotating them, and turning selections into structured raw materials.

## What You Can Do

- Open a source text in read-only mode
- Split the editor into clone or preview view
- Create annotations from selections
- Group annotations into commits and submit them to target folders
