# Source Annotations

Annotations let you turn a selected span of source text into a reusable raw-material entry.

## Workflow

1. Open a source text inside the workspace
2. Select a passage
3. Choose a color and tag
4. Add the annotation to a draft commit
5. Pick a target raw-material folder and submit the commit

## Notes

- Highlight colors help visually group related references
- Marker numbers are kept stable for committed references
- Commits can contain one or multiple annotations
